# Hey, I'm Jonathan 👋

IT guy turned data nerd. I've spent 13 years keeping schools' lights on (digitally speaking) — and somewhere along the way I got really into what the data was saying. Now I'm chasing that full time.

Currently finishing a **Professional Certificate in Applied AI & Data Science** through MIT Professional Education. By day I'm an IT Director. By night I'm training models and breaking things locally so I don't break them in production.

---

## 🛠️ Things I've Actually Shipped

**[E-Rate Scout](https://eratescout.com)** — Full-stack SaaS for navigating the FCC's E-Rate program. Millions of data points, real-time filters, daily change alerts. Yes, it's as niche as it sounds. Yes, people use it.

**[incoGNITO](https://gnito-app.onrender.com)** · [source](https://github.com/jcahal/gnito) — Self-destructing secret messages. Encrypted, deleted on first read. Because sometimes you just need a message to disappear.

**[CSV Data Pipeline](https://github.com/jcahal/csv-pipeline)** — CLI-driven ETL pipeline. Validation, normalization, null removal, encoding. Nothing glamorous, but it works and that's the point.

**[ANSR Ascend Project](http://pc-ascend-team.github.io)** · [source](https://github.com/PC-Ascend-Team) — NASA-funded high-altitude balloon firmware. We sent sensors to near-space and collected more data points than anyone in the program's history. Casual.
